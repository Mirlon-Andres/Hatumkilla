# Hatum Killa Contratistas Generales - Corporate Website

Sitio web corporativo oficial para **Hatum Killa Contratistas Generales E.I.R.L.**, diseñado con una arquitectura **Mobile-First** y altos estándares de rendimiento.

## 🚀 Tecnologías
- **HTML5** Semántico (Atributos ARIA integrados para accesibilidad).
- **CSS3 Vanilla** (Uso de CSS Custom Properties, Flexbox, Grid y tipografía responsiva mediante `clamp()`).
- **JavaScript Moderno (ES6+)** sin librerías externas (Zero jQuery) para máxima velocidad de carga, utilizando `IntersectionObserver` para animaciones y scroll.

## 📁 Estructura del Proyecto
- `index.html`: Estructura principal, dividida por secciones (Hero, Sinergia, Servicios, Respaldo, Clientes, Contacto).
- `css/main.css`: Estilos segmentados por variables, layout responsivo y componentes.
- `js/main.js`: Lógica ligera para navegación inteligente, carruseles y validación superficial de formularios.
- `assets/img/`: Recursos gráficos y fotografías optimizadas.

## 🛠️ Cómo Visualizar
No requiere entorno de Node ni compiladores. Solo abre el archivo `index.html` en tu navegador moderno preferido. Para ver el comportamiento *smart scroll*, asegúrate de navegar hacia abajo y luego hacia arriba.
